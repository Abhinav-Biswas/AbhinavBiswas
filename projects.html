The Augmented Reality Technique

An interactive Virtual mirror (using Webcam) portraying the augmented view of the user in a dynamic Live Video mode with virtual super-imposed clothes.
The vClothing concept empowers the user to try wearing a virtual cloth and have an augmented view of himself before he can actually decide to buy the cloth, after a computerized visualization whether it will really suit him or not.

JavaCV, OpenCV 2.3.1, Swing (Java 1.6),

Ion Mobility Spectrometry

An IMS based Explosive detector can analyze explosive ions by computing the drift time of the ions based on the peak value of the collected ADC sample data and comparing it with the available library data. The IMS application was developed using Qt 4.5 which was ported to the ARM board running embedded Linux.

Technologies involved

JavaCV, OpenCV 2.3.1, Swing (Java 1.6),
Face Detection, Motion Tracking.
